Prerequisites

Before You Start

1

Clone & Install Friday

Friday runs as a separate Bun server. Clone her repo and install dependencies.

git clone https://github.com/byronmcclain/friday.git
cd friday
bun install
2

Get an xAI API Key (Grok Voice)

Friday uses Grok (by xAI) for her voice engine. This is separate from Groq which powers council chat banter.

# 1. Go to:
https://console.x.ai/

# 2. Sign up / sign in
# 3. Navigate to API Keys section
# 4. Click "Create API Key"
# 5. Name it "friday-studio" or similar
# 6. Copy the key (starts with xai-...)
⚠
Grok vs Groq — These are two different services!
Grok (xAI) = Friday's voice engine. Key goes in CoE Settings → Friday section.
Groq (Groq Inc.) = Council Chat banter. Already configured separately.
3

Install the CoE Module into Friday

This symlinks the studio tools into Friday's forge system so she can read your studio data.

# From the CoE project root:
cd C:\Claude\Agency\council-of-elrond
bash friday-module/setup.sh
ℹ
This creates a symlink at ~/.friday/forge/coe-studio/ pointing to your module, and copies the GENESIS identity file if not already present.
Configuration

Connect Friday to CoE

4

Start the Friday Server

Launch Friday in a separate terminal. She listens on port 3000 by default.

# In the friday/ directory:
bun run serve

# You should see:
Friday listening on ws://localhost:3000/ws
5

Start CoE Dashboard

Launch the Council of Elrond Electron app.

# In the council-of-elrond/ directory:
npm run dev
6

Enter xAI Key in Settings

Open the CoE Settings panel and configure Friday.

# In the dashboard:
1. Click the ⚙ Settings gear icon
2. Scroll to "Friday — Studio Director" section
3. Toggle Enable Friday → ON (fuchsia)
4. Server URL: ws://localhost:3000/ws (default)
5. Paste your xAI API key in the "xAI API Key" field
6. Click Save
7

Navigate to Friday Panel

Click "Friday" in the top nav bar (next to "Studio Overview"). The status dot should turn green once connected.

ℹ
If connection fails, check the terminal running Friday for errors. The WebSocket will auto-retry with exponential backoff (1s → 2s → 4s → ... → 30s max, 10 retries).

⚠ Emergency Rollback

If Friday causes issues, you can cleanly revert the entire integration with a single command. All 18 commits were merged via --no-ff, making the merge commit fully reversible.

Option A: Revert (preserves history)

git revert --no-commit HEAD
git commit -m "Revert: Remove Friday Studio Director integration"

Option B: Hard reset (destructive)

# Reset to the commit BEFORE Friday merge:
git reset --hard d18cd9d
⚠
Option B is destructive and will also discard any commits made after the Friday merge. Use Option A unless you're certain.
